Documentation Overview
======================

`ifrs16 <https://github.com/bradley-smith-gpa/ifrs16>`_ is a Python package to make simple lease calculations
with respect to `IFRS 16`_.

.. _`IFRS 16`: https://www.ifrs.org/issued-standards/list-of-standards/ifrs-16-leases/

.. note::

   This project is still in its development phase.
